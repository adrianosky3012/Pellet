# 🔥 Consumi Pellet

App per il tracciamento delle spese invernali di pellet e gas.  
Funziona su **Android**, **iPhone** e **PC** — installabile come app (PWA).

---

## 📁 File del progetto

```
consumi-pellet/
├── index.html       ← app principale
├── manifest.json    ← configurazione PWA
├── sw.js            ← service worker (funziona offline)
├── icons/
│   ├── icon-192.png
│   └── icon-512.png
└── README.md
```

---

## 🚀 Come pubblicare su GitHub Pages

1. Crea un nuovo repository su GitHub (es. `consumi-pellet`)
2. Carica tutti i file mantenendo la struttura delle cartelle
3. Vai su **Settings → Pages**
4. In **Source** seleziona `main` branch, cartella `/ (root)`
5. Clicca **Save**
6. Dopo qualche secondo l'app sarà disponibile su:  
   `https://TUONOME.github.io/consumi-pellet/`

---

## 📱 Installare come app su Android

1. Apri il link con **Chrome**
2. Tocca i tre puntini in alto a destra
3. Seleziona **"Aggiungi a schermata Home"** oppure **"Installa app"**
4. L'app appare come icona sul desktop, si apre a schermo intero

## 🍎 Installare su iPhone/iPad

1. Apri il link con **Safari**
2. Tocca il tasto **Condividi** (quadrato con freccia in su)
3. Seleziona **"Aggiungi a schermata Home"**

## 💻 Usare su PC

Apri il link con **Chrome** o **Edge** — nella barra dell'indirizzo apparirà un'icona  
per installare l'app sul desktop.

---

## 💾 Dati

I dati vengono salvati automaticamente nel browser (localStorage).  
Funziona anche **offline** dopo il primo caricamento.

> ⚠️ I dati sono legati al browser/dispositivo. Per fare un backup,
> usa la funzione copia/incolla dei dati dal menu dell'app.
